# dbx_escape_string()
(PHP 4 >= 4.3.0, PHP 5 < 5.1.0, PECL dbx >= 1.1.0)


dbx_escape_string คือ ฟังก์ชั่นที่ใช้หลีกเลี่ยงสตริง เพื่อให้ใช้คำสั่ง sql-statement ได้ปลอดภัยมากขึ้น

### รูปแบบการใช้งาน
```
dbx_escape_string ( object $link_identifier , string $text ) : string
```
